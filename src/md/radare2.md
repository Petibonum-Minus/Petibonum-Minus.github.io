* ```r2 -d <file>``` : open debug mode
* ```e asm.syntax=att``` : change syntax ('att' or 'intel')
* ```aaa``` : analyse file
* ```afl``` : list all the function run
* ```s <function>``` : focus on function
* ```pdf @<file>``` : open file
